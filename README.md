# Windsurf 8 & 9 Demo Repository

This repository demonstrates the cutting-edge features introduced in Windsurf Waves 8 & 9, showcasing how to leverage these capabilities for efficient software engineering workflows.

## Featured Capabilities

### SWE-1 Model Family Integration
- Demonstrates how to effectively use the SWE-1 family of models:
  - SWE-1: Tool-capable, high-reasoning frontier model
  - SWE-1-lite: Improved foundation model (replacement for Cascade Base)
  - SWE-1-mini: Fast, lightweight passive coding experience

### Custom Workflows
- Reusable, shareable workflows defined in `.windsurf/workflows/`
- Step-by-step instructions that can be invoked with simple commands
- Repository-based sharing for team collaboration

### File-Based Rules
- Multiple rule files in `.windsurf/rules/` with different scopes
- Various activation modes: Manual, Always On, Model Decision, Glob
- Project-specific, team-specific, and organization-wide rules

### Simultaneous Cascades
- Run multiple Cascade conversations concurrently
- Switch between conversations seamlessly
- Background processing with notifications

### MCP Integration
- Model Context Protocol integration for extended capabilities
- Support for MCP resources and multimodal responses
- Integration with external tools and knowledge bases

## Getting Started

1. Clone this repository
2. Open it in Windsurf
3. Explore the examples in each directory
4. Try out the custom workflows with `/workflow-name` commands
5. Observe how the file-based rules affect Cascade's behavior

## Directory Structure

